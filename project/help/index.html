Description

NEXT V-card is an easy cutomizable v-card/portfolio/mini website template made entierly 95% with html and css it allows you to introduce yourself with the most important informations to your features clients or employers.

NEXT V-card can be used also as a mini website for a small compagny that want to be introduced quickly and easly.


The template is responsive that's allows it to view it in the mobile devices also.


All the html ,css, javascript code are well organized and commented to make any change easy to do without any problems.

 

Features

  1. Responsive layout
  2. Retina Ready
  3. Gradient animation background
  4. About section
  5. Resume section
  6. Skills section
  7. Work section
  8. Contact section with a working contact form

Theme installation

When you download the main download file you finde several folder  :

  1. HTML folder this is the folder that contain all the filse of the template
  2. LICENSES folder this folder contain the licenses of the elements used.
  3. DOCUMENTATION : this folder contain the documentation of the theme ,please read the documentation carefully for further information

Gradient background

NEXT vCard come with a cool gradient effect animation in the header and the footer of the theme, you can customize the gradient background with the combination of the color that you want, you cab do that easly by changing the hexadecimal color values in the setup of the gradient background, and you will dinf that in the script.js file unther  Gradient animation section.

 

More info about this cool plugin in the link below :

https://sarcadass.github.io/granim.js/

Icons

NEXT vCard uses fonttello icons , the template come with a set of icons that you can use as you want, and change the default icons to fit your needs.

You can find the list of the icons in the fontello.css in the css  folder then you can choose the icon that you want and change it in the html code

Map setup

NEXT vCard come with a map location section to locate your adress in google map. To setup your coordinate open index.html file then scroll down until you find the map script setup. You are concerned with the lat = -33.867487, lng = 151.20699 values change them to your coordinate location in the map.

Contact form setup

NEXT come whith a good to go ajax contact form ready to use, you have to replace the adress of the recipient in the send.php file by your adresse.

To do that open the send.php file located in the main download file in any editor of your choice and go to the line 34  and change the adress by your adresse.
